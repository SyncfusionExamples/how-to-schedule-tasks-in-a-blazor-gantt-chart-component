# How to Schedule Tasks in a Blazor Gantt Chart Component

Repository Description: A Blazor WebAssembly application that demonstrates how to schedule tasks in a Syncfusion Blazor Gantt Chart component, with support for workweek, holidays, and custom scheduling modes.

The [Blazor Gantt Chart](https://www.syncfusion.com/blazor-components/blazor-gantt-chart?utm_source=github&utm_medium=listing&utm_campaign=blazor-gantt-chart-github-samples) is a project planning and management tool that provides a Microsoft Project-like interface to display and manage hierarchical tasks with timeline details.

## Overview

This quick-start project demonstrates how to schedule tasks in a Syncfusion Blazor Gantt Chart in a Blazor WebAssembly app. Learn to schedule tasks based on workweek, holidays, working time range, and task scheduling modes.

## Prerequisites

* Visual Studio 2022
* .NET 6 or later

## How to Run

1. Clone or checkout this project to your local machine
2. Open the solution in Visual Studio 2022
3. Restore NuGet packages by rebuilding the solution
4. Run the project

## Key Features

- **Data binding** with JSON, RESTful services, and OData sources
- **Timeline views** (hour, day, week, month, year)
- **Task relationships** and dependencies
- **Interactive editing** with dialogs and taskbars
- **Filtering and sorting** capabilities
- **Accessibility** with keyboard navigation and screen reader support
- **Responsive design** with touch support and mobile optimization
- **Globalization** and localization support
