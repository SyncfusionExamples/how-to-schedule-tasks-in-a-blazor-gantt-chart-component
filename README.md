# How to Schedule Tasks in a Blazor Gantt Chart Component

The [Blazor Gantt Chart](https://www.syncfusion.com/blazor-components/blazor-gantt-chart?utm_source=github&utm_medium=listing&utm_campaign=blazor-gantt-chart-github-samples) is a project planning and management tool that provides a Microsoft Project-like interface for visualizing and managing hierarchical tasks on a timeline.

This repository contains a quick‑start sample that demonstrates how to schedule tasks in the Syncfusion Blazor Gantt Chart using a Blazor WebAssembly application. You will learn how to schedule tasks based on the workweek, holidays, working time range, and task scheduling modes in the Gantt Chart.

**Documentation**: https://blazor.syncfusion.com/documentation/gantt-chart/scheduling-tasks   

## Project overview

Task scheduling in the Blazor Gantt Chart is driven by a combination of calendar settings and task constraints. This sample illustrates how the Gantt Chart automatically recalculates task start and end dates when scheduling rules are applied.
The project demonstrates how scheduling behavior adapts when:

- The working calendar changes
- Holidays are defined
- Working hours are restricted
- Task relationships influence execution order

This approach enables accurate planning and timeline visualization for project management applications.

- Clone or download this repository to your local system.
- Open the project file (.csproj) in Visual Studio 2022 or later.
- Restore the required NuGet packages.
- Build the project to ensure there are no compilation errors.
- Run the application.

## Features and benefits

### Task Scheduling

- Schedule tasks based on workweek and working time ranges
- Define holidays and non‑working days
- Automatically recalculate task dates
- Support for multiple scheduling modes

### Timeline
The Blazor Gantt Chart supports different configurable timeline views: hour, day, week, month, and year.

### Task relationships
Create relationships among different tasks to determine the execution order using finish-to-start, start-to-finish, start-to-start, and finish-to-finish task link types.

### Editing
Edit a task's fields directly in their respective cells using a dialog or interactively using taskbars.

### Unscheduled tasks
The Blazor Gantt Chart provides built-in support for unscheduled tasks. These are tasks not scheduled with proper dates or duration at the start of the project but that can be scheduled properly at any time during the project implementation based on factors such as resource availability and dependent tasks.
